## 🎯 How to Get Started

### Step 1: Clone Repository (0 minutes)
Your code is already on GitHub. Just push when you make changes:
```bash
git status  # Should be clean
git log --oneline -5  # Shows all 4 commits
```

### Step 2: Deploy to Render (5 minutes)
1. Go to https://render.com/
2. Connect your GitHub repo
3. Set start command: `python web_app_mpesa.py`
4. Deploy

### Step 3: Get API Keys (15 minutes)
**M-Pesa:**
- Visit: https://developer.safaricom.co.ke/
- Get Consumer Key, Secret, Shortcode, Passkey

**Airtel Money:**
- Visit: https://sandbox.airtel.africa/
- Get Client ID, Secret, Merchant ID

### Step 4: Add Environment Variables (2 minutes)
In Render → Environment Variables:
```
MPESA_CONSUMER_KEY=your_key
MPESA_CONSUMER_SECRET=your_secret
MPESA_SHORTCODE=174379
MPESA_PASSKEY=your_pass
AIRTEL_CLIENT_ID=your_id
AIRTEL_CLIENT_SECRET=your_secret
AIRTEL_MERCHANT_ID=your_merchant
```
