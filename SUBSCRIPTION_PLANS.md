# Trading Bot Subscription Plans

## Available Plans

### 🟢 FREE TIER
- **Price**: $0/month
- **Signal Detection**: Yes (limited to 1 check per hour)
- **Auto Trading**: No
- **Platforms**: Deriv only
- **API Rate Limit**: 100 requests/day
- **Support**: Community only
- **Best for**: Learning, testing, hobbyists

### 🔵 PRO TIER
- **Price**: $49/month (or $490/year - save 2 months)
- **Signal Detection**: Unlimited
- **Auto Trading**: Yes (up to 10 symbols)
- **Platforms**: Deriv + Alpaca
- **API Rate Limit**: 10,000 requests/day
- **Support**: Email support (24-48 hour response)
- **Features**:
  - Email alerts for signals
  - Trade history & analytics
  - Backtesting (limited)
- **Best for**: Active traders, small accounts (<$10k)

### 🟣 ENTERPRISE TIER
- **Price**: $199/month (or $1,990/year - save 2 months)
- **Signal Detection**: Unlimited
- **Auto Trading**: Yes (unlimited symbols)
- **Platforms**: Deriv + Alpaca + Interactive Brokers + TradingView
- **API Rate Limit**: Unlimited
- **Support**: Priority email + Discord channel
- **Features**:
  - Everything in Pro, plus:
  - Advanced backtesting with optimization
  - Custom strategy development
  - Webhook support
  - White-label option
  - API access for integrations
  - Dedicated account manager (monthly check-in)
- **Best for**: Professional traders, hedge funds, developers

### 🏢 DESKTOP APP (ONE-TIME)
- **Price**: $299 (one-time purchase)
- **Features**:
  - Standalone Windows/Mac/Linux executable
  - No internet required for signal detection
  - Local database of trades
  - Offline backtesting
  - All Pro features (lifetime)
  - Future updates (1 year)
- **Limitations**: Single machine license
- **Best for**: Traders who prefer local control

### ☁️ CLOUD MANAGED (PREMIUM)
- **Price**: $99/month (Pro-level) or $299/month (Enterprise-level)
- **Hosting**: Bot runs on our servers 24/7
- **Features**:
  - 99.9% uptime SLA
  - Automatic backups
  - Cloud storage for settings & trades
  - Multi-device access (web dashboard)
  - Mobile app alerts
  - Zero setup required
- **Best for**: Traders who want "set and forget"

## Upgrade Path

Users can start with **Free** → upgrade to **Pro** → upgrade to **Enterprise**

Cloud and Desktop are alternative ways to run the same bot with different trade-offs:
- **Desktop**: Full control, no subscription, but you manage it
- **Cloud**: Zero management, always running, recurring cost
- **Pro/Enterprise**: Your server or local (choose cloud add-on for $50/mo extra)
